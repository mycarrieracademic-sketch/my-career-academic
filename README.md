# My Career Academic - Coaching Management System

## Setup
1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000

## Login
- Admin: admin@mycareeracademic.com / Admin@12345

## Features
- Student Admission & Management
- Live Class Tracking
- Attendance Management  
- Fee Management
- Tests & Results
- Staff Management
- Role-based Access (Admin, Teacher, Student, Guardian)

## Tech Stack
- Next.js 14
- Supabase (PostgreSQL + Auth)
- React
